Variables Aleatorias, Funciones de Densidad y Distribución

Estadística Continua: Variables Aleatorias Continuas

A diferencia de las variables aleatorias discretas, que toman valores específicos (como el número de caras al lanzar una moneda), las variables aleatorias continuas pueden tomar cualquier valor dentro de un intervalo. Ejemplos comunes incluyen la altura de una persona, la temperatura o el tiempo de espera.

Función de Densidad de Probabilidad (PDF)

Para una variable aleatoria continua \(X\), no hablamos de la probabilidad de que tome un valor exacto (esta probabilidad es siempre cero, \(P(X=x)=0\)), sino de la probabilidad de que caiga dentro de un intervalo. Esto se describe mediante la Función de Densidad de Probabilidad (PDF), denotada como \(f_X(x)\).

La PDF debe cumplir dos condiciones:

  1. \(f_X(x) \ge 0\) para todo \(x\).
  2. El área total bajo la curva de la PDF debe ser 1: \(\int_{-\infty}^{\infty} f_X(x) dx = 1\).

La probabilidad de que \(X\) tome un valor en el intervalo \([a, b]\) se calcula integrando la PDF en ese intervalo:

\[ P(a \le X \le b) = \int_{a}^{b} f_X(x) dx \]

Ejemplo (Adaptado de Ejercicio 2.1)

Sea \(X\) una variable aleatoria con PDF:

\[ f_X(x) = \begin{cases} \frac{1}{6}(x+1) & \text{si } 1 < x < 3 \\ 0 & \text{en otro caso} \end{cases} \]

Para calcular \(P(1 < X < 2.5)\):

\[ P(1 < X < 5/2) = \int_{1}^{5/2} \frac{1}{6}(x+1) dx = \frac{1}{6} \left[ \frac{x^2}{2} + x \right]_{1}^{5/2} \] \[ = \frac{1}{6} \left( \left(\frac{(5/2)^2}{2} + \frac{5}{2}\right) - \left(\frac{1^2}{2} + 1\right) \right) = \frac{1}{6} \left( \frac{25}{8} + \frac{5}{2} - \frac{1}{2} - 1 \right) = \frac{1}{6} \left( \frac{25+20-4-8}{8} \right) = \frac{33}{48} = \frac{11}{16} \]
(Nota: El cálculo original en el texto tenía un pequeño error, aquí se presenta corregido).

Función de Distribución Acumulada (CDF)

La Función de Distribución Acumulada (CDF), denotada como \(F_X(x)\), da la probabilidad de que la variable aleatoria \(X\) sea menor o igual a un valor específico \(x\).

\[ F_X(x) = P(X \le x) = \int_{-\infty}^{x} f_X(t) dt \]

Propiedades de la CDF:

  • Es no decreciente: si \(a < b\), entonces \(F_X(a) \le F_X(b)\).
  • \(\lim_{x \to -\infty} F_X(x) = 0\)
  • \(\lim_{x \to \infty} F_X(x) = 1\)
  • Es continua por la derecha (y en el caso continuo, es totalmente continua).

Podemos obtener la PDF derivando la CDF: \(f_X(x) = \frac{d}{dx} F_X(x)\).

La probabilidad en un intervalo también se puede calcular usando la CDF:

\[ P(a < X \le b) = F_X(b) - F_X(a) \]

Ejemplo (Adaptado de Ejercicio 2.3)

Sea \(X\) una variable aleatoria con CDF:

\[ F_X(x) = \begin{cases} 0 & \text{si } x < 0 \\ x^3 & \text{si } 0 \le x < 1 \\ 1 & \text{si } x \ge 1 \end{cases} \]

Esta función cumple las propiedades de una CDF. La PDF se obtiene derivando:

\[ f_X(x) = F'_X(x) = \begin{cases} 3x^2 & \text{si } 0 < x < 1 \\ 0 & \text{en otro caso} \end{cases} \]

Calcular \(P(1/4 < X < 1/2)\):

\[ P(1/4 < X < 1/2) = F_X(1/2) - F_X(1/4) = (1/2)^3 - (1/4)^3 = \frac{1}{8} - \frac{1}{64} = \frac{8-1}{64} = \frac{7}{64} \]

Probabilidad Condicionada

La probabilidad condicionada \(P(A|B)\) se define como la probabilidad de que ocurra el suceso A, sabiendo que ha ocurrido el suceso B. Para variables continuas, si \(A = \{X \in I_A\}\) y \(B = \{X \in I_B\}\), donde \(I_A\) e \(I_B\) son intervalos o conjuntos de valores:

\[ P(A | B) = \frac{P(A \cap B)}{P(B)} = \frac{P(X \in I_A \cap I_B)}{P(X \in I_B)} \]

Donde \(P(B)\) debe ser mayor que cero.

Ejemplo (Adaptado de Ejercicio 2.2)

Sea \(X\) con PDF \(f_X(x) = \frac{1}{2}(x-1)\) si \(1 < x < 3\), y 0 en otro caso. Sean \(A = \{X > 5/4\}\) y \(B = \{X < 5/2\}\). Calcular \(P(A|B)\).

Necesitamos \(P(A \cap B) = P(5/4 < X < 5/2)\) y \(P(B) = P(1 < X < 5/2)\) (ya que la PDF es 0 para \(x \le 1\)).

\[ P(A \cap B) = \int_{5/4}^{5/2} \frac{1}{2}(x-1) dx = \left[ \frac{1}{4}(x-1)^2 \right]_{5/4}^{5/2} = \frac{1}{4} \left( (\frac{3}{2})^2 - (\frac{1}{4})^2 \right) = \frac{1}{4} \left( \frac{9}{4} - \frac{1}{16} \right) = \frac{1}{4} \left( \frac{36-1}{16} \right) = \frac{35}{64} \] \[ P(B) = \int_{1}^{5/2} \frac{1}{2}(x-1) dx = \left[ \frac{1}{4}(x-1)^2 \right]_{1}^{5/2} = \frac{1}{4} \left( (\frac{3}{2})^2 - (0)^2 \right) = \frac{1}{4} \cdot \frac{9}{4} = \frac{9}{16} = \frac{36}{64} \] \[ P(A | B) = \frac{35/64}{36/64} = \frac{35}{36} \]

Valor Esperado (Esperanza Matemática)

El valor esperado o media de una variable aleatoria continua \(X\), denotado como \(E[X]\) o \(\mu\), representa el valor promedio que tomaría \(X\) si repitiéramos el experimento muchas veces.

\[ E[X] = \int_{-\infty}^{\infty} x f_X(x) dx \]

Para una función \(g(X)\) de la variable aleatoria \(X\), el valor esperado es:

\[ E[g(X)] = \int_{-\infty}^{\infty} g(x) f_X(x) dx \]

Una propiedad importante es la linealidad de la esperanza: \(E[aX + b] = aE[X] + b\).

Ejemplo (Adaptado de Ejercicio 2.4)

Sea \(X\) con PDF \(f_X(x) = 4(1-x)^3\) si \(0 \le x < 1\), y 0 en otro caso. Calcular \(E[X]\).

\[ E[X] = \int_{0}^{1} x \cdot 4(1-x)^3 dx \]

Se puede resolver por partes o desarrollando \((1-x)^3\). El resultado es:

\[ E[X] = \frac{1}{5} \]

Ejemplo (Adaptado de Ejercicio 2.7)

Sea \(X\) con PDF \(f_X(x) = 2x\) si \(0 < x \le 1\), y 0 en otro caso. Calcular \(E[X^2+1]\).

\[ E[X^2+1] = \int_{0}^{1} (x^2+1) \cdot 2x dx = \int_{0}^{1} (2x^3 + 2x) dx = \left[ \frac{2x^4}{4} + \frac{2x^2}{2} \right]_{0}^{1} = \left[ \frac{x^4}{2} + x^2 \right]_{0}^{1} = \frac{1}{2} + 1 = \frac{3}{2} \]

Alternativamente, \(E[X^2+1] = E[X^2] + 1\). Calculamos \(E[X^2] = \int_{0}^{1} x^2 \cdot 2x dx = \int_{0}^{1} 2x^3 dx = [\frac{2x^4}{4}]_0^1 = \frac{1}{2}\). Entonces \(E[X^2+1] = \frac{1}{2} + 1 = \frac{3}{2}\).

Transformaciones de Variables Aleatorias

Si tenemos una variable aleatoria \(X\) con PDF \(f_X(x)\) y creamos una nueva variable \(Y = g(X)\), ¿cuál es la PDF de \(Y\), \(f_Y(y)\)?

Un método general es encontrar primero la CDF de \(Y\), \(F_Y(y) = P(Y \le y) = P(g(X) \le y)\), expresando la condición en términos de \(X\), y luego derivar para obtener \(f_Y(y)\).

Ejemplo (Adaptado de Ejercicio 2.16)

Sea \(X\) con PDF \(f_X(x) = x/8\) si \(0 \le x \le 4\), y 0 en otro caso. Sea \(Y = \sqrt{X}\). Hallar \(f_Y(y)\).

Como \(X\) varía entre 0 y 4, \(Y = \sqrt{X}\) varía entre 0 y 2.

Para \(0 \le y \le 2\):

\[ F_Y(y) = P(Y \le y) = P(\sqrt{X} \le y) = P(X \le y^2) \] \[ = \int_{0}^{y^2} \frac{x}{8} dx = \left[ \frac{x^2}{16} \right]_{0}^{y^2} = \frac{(y^2)^2}{16} = \frac{y^4}{16} \]

La CDF completa es:

\[ F_Y(y) = \begin{cases} 0 & \text{si } y < 0 \\ y^4/16 & \text{si } 0 \le y \le 2 \\ 1 & \text{si } y > 2 \end{cases} \]

Derivando para obtener la PDF:

\[ f_Y(y) = F'_Y(y) = \begin{cases} \frac{4y^3}{16} = \frac{y^3}{4} & \text{si } 0 < y < 2 \\ 0 & \text{en otro caso} \end{cases} \]
(Nota: El texto original tenía \(1/4 y^3\) en el intervalo \(0 \le y \le 2\), lo cual es correcto).

Distribución Normal

Una de las distribuciones continuas más importantes es la Distribución Normal (o Gaussiana), denotada como \(X \sim N(\mu, \sigma^2)\), donde \(\mu\) es la media y \(\sigma^2\) es la varianza (\(\sigma\) es la desviación típica).

Su PDF es:

\[ f(x; \mu, \sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \]

Como la integral de esta función no tiene una primitiva elemental, los cálculos de probabilidad se realizan mediante la tipificación (o estandarización) de la variable. Se transforma \(X\) en una variable \(Z\) con distribución normal estándar \(N(0, 1)\):

\[ Z = \frac{X - \mu}{\sigma} \]

Las probabilidades para \(Z\) se encuentran en tablas estándar o se calculan con software. La CDF de \(Z\) se suele denotar como \(\Phi(z)\).

Propiedades útiles:

  • \(P(X \le a) = P\left(Z \le \frac{a-\mu}{\sigma}\right) = \Phi\left(\frac{a-\mu}{\sigma}\right)\)
  • \(P(X > a) = 1 - P(X \le a) = 1 - \Phi\left(\frac{a-\mu}{\sigma}\right)\)
  • \(P(a < X \le b) = P\left(\frac{a-\mu}{\sigma} < Z \le \frac{b-\mu}{\sigma}\right) = \Phi\left(\frac{b-\mu}{\sigma}\right) - \Phi\left(\frac{a-\mu}{\sigma}\right)\)
  • Por simetría de \(N(0,1)\): \(\Phi(-z) = 1 - \Phi(z)\)

Ejemplo (Adaptado de Ejercicio 2.6)

Sea \(X \sim N(1, 4)\) (media \(\mu=1\), varianza \(\sigma^2=4\), desviación típica \(\sigma=2\)). Calcular \(P(X > 2 | X < 3)\).

\[ P(X > 2 | X < 3) = \frac{P(2 < X < 3)}{P(X < 3)} \]

Tipificamos:

\[ P(X < 3) = P\left(\frac{X-1}{2} < \frac{3-1}{2}\right) = P(Z < 1) = \Phi(1) \] \[ P(2 < X < 3) = P\left(\frac{2-1}{2} < \frac{X-1}{2} < \frac{3-1}{2}\right) = P(0.5 < Z < 1) = \Phi(1) - \Phi(0.5) \]

Usando valores típicos de la tabla normal estándar (\(\Phi(1) \approx 0.8413\), \(\Phi(0.5) \approx 0.6915\)):

\[ P(X < 3) \approx 0.8413 \] \[ P(2 < X < 3) \approx 0.8413 - 0.6915 = 0.1498 \] \[ P(X > 2 | X < 3) \approx \frac{0.1498}{0.8413} \approx 0.1781 \]
